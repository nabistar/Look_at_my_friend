## 내 친구를 봐줘!
자랑하고 싶은 사진을 익명으로 자랑할 수 있는 웹 사이트입니다.
typescript, React, Node.js, mySQL을 사용했습니다.

## 주요 기능

1. 원하는 사진과 사진에 대한 설명, 혹은 자랑글을 올릴 수 있도록 구현.
2. 데이터베이스에 올라간 게시글들을 목록화 하여 볼 수 있도록 구현.
3. 게시글 작성 당시 입력한 비밀번호를 입력하면 게시글을 삭제할 수 있도록 구현.
